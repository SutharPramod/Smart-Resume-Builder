## 🎯 **Features**
| Export Format | Status | Features |
|---------------|--------|----------|
| **PDF** | ✅ | Print-ready, professional |
| **HTML** | ✅ | Interactive links, web-ready |

- **Live Preview** with click-to-edit
- **Responsive Design** (desktop/mobile)
- **Social Link Integration** (GitHub/LinkedIn/Twitter/Portfolio)
- **Rich Sections**: Experience, Projects, Education, Skills, Certifications, Summary
- **Sample Data** loaded in 1-click

## 🎮 **Quick Start**

## 🛠 **Tech Stack**
```
Frontend: React 19 + TailwindCSS 4 + Vite 8
PDF: @react-pdf/renderer
Animations: Framer Motion
Icons: React Icons
```

## 📂 **Project Structure**
```
src/components/
├── ExportButtons.jsx (PDF/HTML)
├── Preview.jsx (Live editing)
├── ResumePDF.jsx (PDF renderer)
└── exportHTML.js (HTML utility)

src/components/sections/
├── PersonalSection.jsx

├── SummarySection.jsx
├── ExperienceSection.jsx
├── EducationSection.jsx

```
